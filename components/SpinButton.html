# Form Spinbutton

The Form SpinButton allows the user to adjusting a numeric range with finite control.

# Overview

The component <b-form-spinbutton> is WAI-ARIA compliantopen in new window, allowing for keyboard control, and supports both horizontal (default) and vertical layout.

<b-form-spin-button  min="1" max="100" step="1" />

<script lang='ts' setup>
  import {ref, computed} from 'vue'
  const ex1Value = ref(50);
</script>

The ArrowUp and ArrowDown keys can be used to increment or decrement the value.
